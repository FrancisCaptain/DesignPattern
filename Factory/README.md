### 产品族和产品等级结构之间的关系

产品等级：抽象某一类别的产品，派生出这一类别的产品。例：鞋子，阿迪达斯的鞋子、耐克的鞋子、匡威的鞋子等，鞋子为抽象中的类别，具体的品牌为其子类。

产品族：不同产品等级下面不同类别的一组产品。例：阿迪达斯下面可以有篮球、衣服、鞋子等，由同一工厂创建出来的同品牌不同类别的产品。
